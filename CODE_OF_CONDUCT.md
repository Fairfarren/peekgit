# 行为准则

我们希望 `peekgit` 社区保持专业、友善、开放。

## 我们鼓励的行为

- 尊重不同背景和观点
- 以建设性方式提出反馈
- 接受并讨论合理的技术分歧
- 对新贡献者保持耐心和支持

## 不可接受的行为

- 人身攻击、侮辱、骚扰或歧视性言论
- 恶意挑衅、持续性破坏讨论秩序
- 未经同意公开他人隐私信息
- 其他明显不利于社区协作的行为

## 执行机制

维护者有权对违规内容进行编辑、删除、锁定讨论，并视情节采取警告、限制互动或封禁等措施。

## 报告方式

如果你遇到不当行为，请通过仓库 Issue 或维护者联系方式反馈，并尽量附上上下文。

我们会在合理时间内处理，并尽量保护反馈者隐私。
